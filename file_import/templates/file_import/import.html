The name of each file within your ZIP archive should uniquely identify an instance of the destination model you specify. For example, if you selected a model called Flavor and an update key called name, and you uploaded a ZIP archive containing a file called chocolate.png, the importer would attempt to find an instance of Flavor with name equal to chocolate. Capitalization of the file name does not matter.
